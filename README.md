# k2gehooker
An alternative hooker to MSHookFunction for jailed devices based on breakpoints, yet keeps a capability of calling original functions!

# Usage
Full examples are available in the `Tweak.xm`!

# Notes
- works on both jailbroken and non-jailbroken (sideloadable)
- limited to 6 hooks at the same time
